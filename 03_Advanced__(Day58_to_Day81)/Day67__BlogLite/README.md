## <p align="center"> BlogLite - Mini CMS de blog avec Flask </p>
### <p align="center"> #Day67 </p>

#### Aperçu
**BlogLite** est une application web minimaliste développée avec **Flask** permettant de créer, éditer, consulter et supprimer des articles de blog via une interface simple. Ce projet introduit l'utilisation de **Flask-WTF**, **SQLAlchemy**, **CKEditor**, et de **Bootstrap** pour structurer une application CRUD complète avec persistance de données.

#### Compétences acquises
- **Flask** : Routes, templates Jinja, gestion des formulaires, redirections.
- **ORM SQLAlchemy** : Modélisation, création et manipulation de base de données.
- **Formulaires avec Flask-WTF** : Champs avec validations, soumissions conditionnelles.
- **Base de données SQLite** : Connexion, création de tables, requêtes, suppressions.
- **CKEditor** : Intégration d’un éditeur de texte enrichi pour les articles.
- **Bootstrap5** : Design responsive et moderne.

#### Partie obligatoire
- **Pages disponibles** : Accueil avec liste des articles, page de lecture individuelle, formulaire d'ajout, formulaire d’édition, bouton de suppression.
- **Modèle de données** : Table `BlogPost` avec titre, sous-titre, auteur, image, contenu, date.
- **Formulaire** : Création et édition d'article via un formulaire structuré avec validations.
- **Affichage dynamique** : Articles affichés depuis la base de données avec rendu HTML.
- **Navigation** : Pages "About" et "Contact" statiques.

#### Pourquoi ce projet est pertinent
**BlogLite** est un excellent point d'entrée dans le développement web backend avec Flask. Il combine la manipulation de données via SQLAlchemy, la validation côté serveur, et l’intégration de composants frontend modernes. C’est un projet complet, réutilisable et extensible, typique d’une application CRUD réelle, et formateur pour les futurs projets plus complexes.
